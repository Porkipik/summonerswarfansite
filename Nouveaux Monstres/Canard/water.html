Canard Eau : Carleau

Carleau est le Canard Eau. Il est de type Attaque et ses compétences offensives détruiront tous les ennemis devant vous!


Compétences :

Comp. 1 : Charge! : Fonce sur un ennemis avec 25% de chances de l'assomer.

Comp. 2 : Position Offensive : Appel tes frères canards pour attaquer un ennemis. Tour de temps de pause : 3 tours.

Comp. 3 : Famille offensive "passif" : 10% de chances qu'un canard attaque le monstre qui vient de t'attaquer.
